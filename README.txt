PRODUCT REQUEST FORM - MOBILE PWA

How to test:
1. Extract the ZIP.
2. Open index.html with Chrome or Edge.
3. For the best mobile experience, upload these files to GitHub Pages or another static host.
4. On Android Chrome: tap the three dots > Add to Home screen.
5. On iPhone Safari: tap Share > Add to Home Screen.

Current version:
- 3D preview is removed.
- Product selection uses Product Family / Product Group / Product Sub Group.
- Pergola groups: Falcate, Subulate, Unifoliate, Pergo Rise.
- Falcate / Subulate / Unifoliate sub groups: Minima, Tectona.
- Pergo Rise has no sub group and shows Yok.
- Bioclimatic group is Bio-Rise and has no sub group.
- B-Cube sub groups include Galaxy, Space, Galaxy Manuel, Freedom, Classic, Infinity, Urban, Freedom Plus.
- Language selector changes the on-screen PDF preview language.
- Created/shared PDF output remains English.

No server, no login, no database. User data is saved locally on the device.
